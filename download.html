Verify the Android download

Check the APK against its published SHA-256 before you install it.

sha256sum -c SHA256SUMS.txt
CertUtil -hashfile VERAQIS-Archiver-Android-1.7.0-rc1.apk SHA256

The signed APK is paired with a SHA-256 checksum in SHA256SUMS.txt. A checksum proves file integrity; it is not a publisher signature. The Android APK is additionally signed with the permanent VERAQIS Archiver release key.

Why packages may be withdrawn

Recovery and ColdPass will return only as fresh, reviewed artifacts. Withdrawing a package is a safety step, not a claim that a previous file was harmful.
