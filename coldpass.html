Archived experimental project

ColdPass is an archived VERAQIS experiment.

ColdPass was an offline, local-vault password manager experiment. It is no longer part of the active VERAQIS product line — there is no active public release or development. This page is kept only as project history.

Archived · no active development

Archived project. The previous v1.0.0 package is no longer distributed, and no new ColdPass binary is planned. Do not install or redistribute previously downloaded copies as a current VERAQIS release. The material below is historical reference only.
LOC

Local by design

The encrypted vault stays on your computer. ColdPass has no cloud account, telemetry or call-home.

AES

Encrypted vault

AES-GCM-256 protects vault contents. The previous implementation derived keys with PBKDF2-SHA-256.

SAFE

Safer storage workflow

Atomic saves and local backups reduce the chance that an interrupted write destroys the current vault.

About the previous v1.0.0 release

These are the features of the earlier, now-withdrawn ColdPass v1.0.0 Windows package — described here for reference, not offered as a current download.

  • Native Windows GUI, self-contained for x64.
  • Create, unlock, search and edit a local encrypted password vault.
  • English and Russian interface, light/dark presentation and accessibility support.
  • Import paths documented for Google/Chrome CSV and JSON in that release.
  • Offline release manifest and ECDSA P-256 Trust Pack verification.
  • That executable was not Authenticode-signed. A SHA-256 and an offline Trust Pack are integrity evidence — they do not replace Windows publisher reputation.

Newer development work is not advertised here until it becomes a clean, verified release.

ColdPass status

ColdPass is archived. It is not part of the active VERAQIS product line, has no active development, and no download will be published. The source code and history are preserved.

Is ColdPass still being developed?

No. ColdPass is an archived experimental project — no active public release or development.

Can I download it now?

No. No ColdPass package is offered for download, and no new binary is planned.

Was the code deleted?

No. The source code and project history are preserved; only the active product status and downloads were retired.

What is VERAQIS focused on instead?

Two active products: VERAQIS Recovery (commercial) and the free VERAQIS Archiver for Android.
