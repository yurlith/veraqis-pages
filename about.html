About VERAQIS

We build proof-first safety tools for files — software that earns trust by showing its work, not by making promises.

What we believe

Data-recovery software has a credibility problem: too many tools "recover" bytes they can't vouch for, and present a guess as a result. VERAQIS takes the opposite stance. The unit we ship is provably-recovered data at zero over-claim. If the surviving bytes prove a result, we return it with its evidence. If they don't, we say so. Unknown is an honest answer.

Our principles

  • No invented data — nothing is emitted unless surviving bytes prove it.
  • Inferred is never labelled verified; a guess never improves a health score.
  • Encryption is detected and abstained — never broken or bypassed.
  • Offline-first: read-only source, sandboxed processing, no telemetry.
  • Private development repositories: only reviewed release artifacts and public documentation are published.

What VERAQIS is not

It is not a password cracker, a DRM remover, or a "recover everything" miracle. It does not give financial advice, and it makes no guarantee that any particular file can be recovered. It is a careful, honest toolkit — currently in beta.

VERAQIS is an independent software project developed by Yury Talkanitsa in Switzerland. It is not a registered company, and no legal entity is implied. The current focus is the beta: VERAQIS Archiver (Android, available now), VERAQIS Recovery (rebuild pending), and Reports & Verification.
