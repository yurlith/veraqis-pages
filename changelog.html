Changelog / Release Notes

An honest record of what's shipped. Dates are when the work landed in the build, not marketing announcements.

Unreleased — Protect-first in source, next build

2 July 2026 · not yet in the hosted v0.2.1 artifacts

  • Folder-level Shield: veraqis protect shield <dir> --recursive protects a whole tree (a parity sidecar per file + a folder manifest), with incremental --update re-shielding only changed files. Gated to the future Pro tier in code; single-file Shield stays free.
  • veraqis protect check — free, read-only rot detection over a protected folder (per-shard damage localization, repairable / not-repairable verdict per file); and veraqis protect repair — free, reconstructs damaged or missing protected files byte-exact into a separate output folder, verified against the original's SHA-256, never in place, never unproven bytes.
  • VERAQIS Recovery gains a Protect screen (protect a folder · check · repair) driving the same engine through the same policy table as the CLI.
  • VERAQIS Archiver (Android) can now verify a file against its .phxs sidecar on-device — a golden-tested port of the engine's damage scan (verify-only; reconstruction stays on desktop/CLI, and the app says so honestly).

Public-beta access update current source

18 July 2026

  • Every current VERAQIS Recovery capability is temporarily free during public beta so early users can provide feedback and measured recovery results.
  • No account, checkout or license file is required for structural recovery, semantic recovery, salvage, reconcile, streaming, Shield or experimental exact-copy recovery.
  • The offline license implementation remains in development for a later commercial release; activating paid enforcement requires a separate release decision.

Commercial-boundary implementation superseded beta plan

18 July 2026

  • The offline entitlement model was implemented with analysis, diagnosis, verification, safe inspection and safe extraction in its permanent Free scope.
  • The planned commercial policy grouped structural, semantic, salvage, reconcile, streaming and exact-copy recovery behind one entitlement gate. Public-beta free access now overrides that plan until a separately approved commercial release.
  • Recovery from an existing Shield sidecar remains Free so protected customer data is never held hostage. VERAQIS Archiver and ColdPass remain separate free products.

v0.2.1 — public beta

26 June 2026; Linux CLI artifact added 29 June 2026

  • First public-beta launch surface: the VERAQIS site now documents the mobile, desktop and verification products honestly, with pricing that matches what the code enforces.
  • Workspace and apps aligned at version 0.2.1; the non-Android release path passed Rust tests, formatting, clippy, Desktop UI audit, Desktop bundle creation, Linux CLI smoke testing and release hash checks.
  • Licensing clarified: only exact-copy reconstruction is gated to a future paid tier; everything else, including Tier-3 semantic recovery, is free during beta.

0.2.0 — proof-carrying engines & mobile

  • Proof-carrying infrastructure: the verified-recovery benchmark, self-contained certificates, a standalone re-verifier that shares no solver code, and a formally-checked emission invariant.
  • 7-Zip & RAR: first-class detection, byte-exact 7z signature and start-header repair, verified inventory and intact-folder / STORED extraction. Solid-LZMA and proprietary RAR codec payloads recorded as a measured NO-GO.
  • VERAQIS Archiver (mobile): safety-scan-before-open, safe listing and SAF-scoped extraction, built on an Android-free safety facade and routed through a single audited seam.
  • Localization: 7 interface languages including right-to-left Arabic.

Earlier milestones

  • Tier-3 semantic recovery for SQLite, OPC/OOXML and PDF — verified rows, parts and objects, gated at zero false results.
  • Partial-decompression salvage (bzip2 / xz) and intra-file redundancy mining (ISO 9660 / ZIP).
  • Format-aware structural repair for ZIP, TAR, GZIP and ISO, plus the GZIP/TAR/DEFLATE resync track.
  • VERAQIS Shield: structure-weighted Reed-Solomon parity protection with a signed sidecar.
  • The free, read-only auditor and the evidence engine that underpins everything: rule-keyed findings, health and recoverability scoring, and the sandboxed trust boundary.
